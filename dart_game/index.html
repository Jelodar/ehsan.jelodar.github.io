Info

This game was developed using Artificial Intelligence, specifically a Genetic Algorithm (GA). The placement of balloons is optimized by the algorithm to ensure the best possible performance and guarantee that all levels are passable, avoiding any unwinnable scenarios.
Through this approach, I have achieved a solution with 100% passability across all levels of the game.
